# 智能算法学习助手

> 基于个人模板库的算法竞赛学习系统——模板查询、AI 题图分析、模板入库维护、学习诊断与反馈，一站式支撑算法学习闭环。

## 项目简介

本项目围绕算法竞赛学习中的真实痛点，将个人积累的 C++ 算法模板库转化为一个**可查询、可理解、可维护**的网页学习系统。

在日常刷题中，常见的问题不是"不会写代码"，而是：

- 面对题目时难以快速判断它属于哪类算法
- 模板分散在本地文件夹，查找效率低
- 即使找到模板，也不清楚适用条件、常见误区和改造方向
- 做完题后无法系统性地追踪自己在哪些算法类别上反复出错

本项目正是为了解决这些问题而设计的。

## 核心功能

### 1. 模板库查询系统

- **分层分类浏览**：将模板按基础算法、数据结构、图论、动态规划等专题组织，支持按分类目录逐级展开
- **名称/标签检索**：输入模板名、目录名或标签关键字，实时过滤匹配结果
- **源码高亮展示**：在网页中直接查看 C++ 模板代码，带语法高亮
- **模板元信息**：每个模板展示难度、适用题型、识别信号、常见风险、关键词等人工整理的说明信息
- **最近查看**：自动记录浏览历史，方便快速回看
- **收藏模板**：收藏常用模板，从侧边栏快速访问

### 2. AI 模板讲解

- 选中模板后按需触发，系统读取模板标题、分类、标签和源码，自动识别算法主题
- 从核心思路、适用场景、识别信号、常见误区、阅读建议等维度生成讲解
- API Key 仅存在当前标签页的 session 中，不写入项目文件

### 3. 题目 AI 分析

- 上传题目图片或粘贴截图，多模态模型理解题意并提取信号
- 从本地模板库中推荐最匹配的候选模板，而非直接生成题解
- 给出算法归类判断、推荐模板及理由
- **用户选择验证**：分析前可先输入你猜测的模板名，AI 会判断你的选择是否正确并给出个性化反馈
- **学习诊断**：基于历史分析记录，自动统计各算法专题的正确率，识别掌握与薄弱专题
- **AI 深度诊断报告**：一键生成包含整体评估、强项、弱项、建议、趋势的结构化学习诊断报告

### 4. 模板入库与维护

- 支持粘贴代码或上传 cpp 文件，AI 智能分类后自动建议文件名和存放目录
- 通过脚本自动扫描本地模板文件，生成网页可用的结构化数据
- 支持在查询页直接删除模板，自动清理对应 AI 元数据和空目录
- AI 辅助元数据维护（`scripts/ai-maintain-template-metadata.mjs`）

## 模板库概览

当前模板库涵盖以下算法专题：

| 专题 | 内容 |
|------|------|
| 基础算法 | 高精度运算、三分法、NTT、字符串（BWT）、排序技巧 |
| 搜索 | BFS、DFS 全排列 |
| 二分 | 整数/实数二分查找、二分答案 |
| 数据结构 | 线段树（区间和/最值/懒标记）、树状数组（单点/区间修改查询）、波兰式求解 |
| 图论 | 并查集、链式前向星、拓扑排序(Kahn)、Dijkstra、Floyd、Kruskal、Prim、LCA |
| 动态规划 | 01背包、完全背包、多重背包、分组背包、依赖背包、LIS、区间DP、树形DP、数位DP、状压DP、斜率优化DP、换根DP 等 |
| 计算几何 | 最小覆盖圆 |

## 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) (v16 或更高)
- 现代浏览器（Chrome / Edge / Safari）

### 启动方式

**方式一：macOS 一键启动（推荐）**

双击项目根目录下的 `启动网页.command` 文件，自动启动本地服务器并打开浏览器。

**方式二：命令行启动**

```bash
# 启动 Web 服务器
node scripts/web-server.mjs

# 浏览器访问
# http://localhost:8123
```

**方式三：重新生成模板数据**

如果你更新了模板库中的 C++ 文件，需要重新生成数据：

```bash
node scripts/generate-template-data.mjs
```

## 设计理念

本项目不是简单的"AI 直接给答案"工具，而是围绕**个人知识库**构建的持续学习支持系统：

1. **以个人模板库为基础**：所有内容源自自己积累的算法模板，确保贴合个人学习进度
2. **结构化组织**：通过人工分层分类 + 标签体系，让模板不仅仅是代码仓库，更是可检索的知识网络
3. **渐进增强**：先搭建查询系统，再逐步叠加 AI 分析与学习闭环能力
4. **离线可用**：核心查询功能不依赖后端服务，纯前端即可运行
